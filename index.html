What is CSS?

  1. CSS stands for Cashcading Style Sheets
  2. CSS describes how HTML element are to be displayed on the screen,paper,or in other media.
  3. Css is not neccesary CSS saves a lot of work. It can control the layout of multiple web pages all at once.
  4. External stylesheeta are stored in CSS files.

CSS can be added to HTML document in 3 ways:

  • Inline-by using the style attribute inside HTML element
  • Internal-by using style element in the head section
  • External-by using a link element to link to an external Css files.
  • The most common way to add CSS,is to keep the styles in external CSS files.However,in this tutorial we will use inline and external styles,because this is easier to demonstrate,and easier for you to try it yourself.

When comparing CSS class vs ID,the difference is the CSS class appliesa style to multiple elements.ID,on the other hand,applies a style to one unique element.ID is also special in that you can use a special URL to link directly to an element and it's used by JavaScript.


In CSS,selectors are used to target a specific alement or range of element on a web page.Once an element has been targeted,a style or set of styles can be applied to the element.

There is a wide range of selectors available.Two of the most commonly used are class and ID.Both are used to target elements to which a style should be applied.
